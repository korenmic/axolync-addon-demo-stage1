# Axolync Demo Stage 1 Addon

This repo owns the deterministic packaged Stage 1 demo addon for Axolync.

It remains intentionally demo-named in this first extraction so its scope stays honest:

- packaged proof addon
- deterministic SongSense / SyncEngine / LyricFlow / SongMetadata adapters
- demo-owned assets and package generation logic

This repo is not the browser host.
Browser and builder consume the addon produced here.
